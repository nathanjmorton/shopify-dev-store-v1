# Shopify Dev Store

### Ramp in steps

1. Create a partner account. my link for example is [here](https://partners.shopify.com/4459918)
2. Create a dev store on the dashboard. my link for example is [here](https://dev.shopify.com/dashboard/180584824/stores)
3. Login to the store on your admin page. my link for example is [here](https://admin.shopify.com/store/dev-store-v1-3/themes?appLoadId=ecc8e221-f8bf-4a52-9cb6-81d717095fe6)
4. Integrate github on the Online Store > Theme view
5. Choose the theme repo and branch from the integration
6. Publish the github repo branch theme
  - now anything you push to that branch will deploy to prod (your published theme)
